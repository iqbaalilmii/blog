# Halo Dunia!

Selamat datang di blog saya! Ini adalah postingan blog pertama saya yang ditulis dalam format Markdown. 

Di sini saya akan membagikan berbagai tulisan, catatan belajar, write-up CTF, atau hal-hal menarik lainnya.

## Kenapa Blog Ini Dibuat?

Blog ini dibuat untuk mendokumentasikan perjalanan belajar saya dan berbagi pengetahuan kepada teman-teman semua. Semoga isi blog ini bisa bermanfaat!

## Menambahkan Gambar

Jika Anda ingin menambahkan gambar pada postingan, Anda bisa:
1. Membuat folder baru di dalam folder `blog-posts/` (misalnya `blog-posts/gambar-post-pertama/`).
2. Masukkan file gambar Anda ke dalam folder tersebut (misal `foto.png`).
3. Panggil di dalam file Markdown menggunakan format:
   ```markdown
   ![Deskripsi Gambar](/gambar-post-pertama/foto.png)
   ```
4. Jalankan `npm run dev` atau `npm run build`, maka gambar tersebut akan otomatis disalin ke folder `public/`.
